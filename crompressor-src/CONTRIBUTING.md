# Contributing to Crompressor

Thank you for your interest in contributing to Crompressor!

## How to contribute

1. Fork and create a branch from `main`.
2. Run `make test` — all tests must pass.
3. Add tests for any new behavior.
4. Open a PR with a clear description of what changes and why.

## Code style

- Follow standard Go conventions (`gofmt`, `go vet`).
- Keep functions focused and testable.
- Write descriptive commit messages.

## Reporting issues

- Use [GitHub Issues](https://github.com/MrJc01/crompressor/issues).
- Include Go version, OS, and steps to reproduce.

## Questions about the math

Questions about the compression model and mathematical foundations belong in
[crompressor-matematica](https://github.com/MrJc01/crompressor-matematica).
